# Playwright 101 Assignment

## Installation

```bash
npm install
```

## Install Playwright Browsers

```bash
npx playwright install
```

## Run Tests

```bash
npm test
```

## Run Headed

```bash
npm run test:headed
```

## View Report

```bash
npm run report
```

## Features

- Parallel Execution
- Chromium + Firefox
- Screenshots
- Videos
- Trace Logs
- Assertions
- Multiple Locator Strategies

```bash
tests/

utils/
```
